MapGenerator
============

Monogame/C#. Generates random maps/paths.

###Instructions  
Space - generate map  
Up/Down - navigate settings  
Enter - change settings  
&nbsp;&nbsp;&nbsp;&nbsp;Up/Down - change by 1  
&nbsp;&nbsp;&nbsp;&nbsp;Left/Right - change by 10  
WASD - move camera  
Page Down/Page Up - zoom in/out
